# Property Listing Feature - Database Schema & Architecture

## Database Schema and Project Structure

**Date:** 2025-11-18
**Status:** Planning Phase
**Client:** Property Listing Feature

---

ERD Sequelize:

```ts
// Property model
Property: {
  title: string
  description: text
  price: decimal
  address: string
  city: string
  state: string
  country: string
  zip: string
  bedrooms: integer
  bathrooms: decimal
  sqft: integer
  amenities: JSON
  status: enum
  createdAt/updatedAt: timestamps
  ownerId: string
}

// PropertyImage model
PropertyImage: {
  id: integer
  propertyId: foreignKey
  filename: string
  url: string
  isPrimary: boolean
  createdAt: timestamp
}

// One-to-many relationship
Property.hasMany(PropertyImage, { foreignKey: 'propertyId' })
PropertyImage.belongsTo(Property)
```
