### 2. 🟠 Event Storming (Backend Input)

```yaml
Actors:
  - Name: User
    Role: Product Manager
    Description: The person inputting product details and managing listings in Shopify.

Commands:
  - Name: CreateProduct
    TriggeredBy: User
    Pre: none
    Next: ProductCreated

  - Name: GenerateImage
    TriggeredBy: User
    Pre: ClientImageUploaded
    Next: ImageGenerated

Events:
  - Name: ProductCreated
    Pre: CreateProduct
    Next: ProductListingVisible

  - Name: ImageGenerated
    Pre: GenerateImage
    Next: ProductUpdatedWithImage

Policies:
  - Name: ProductInputValidation
    Notes: Ensure all required product fields are filled before submission.
    Pre: CreateProduct
    Next: CreateProduct

External systems:
  - Name: ShopifyAPI
    Pre: CreateProduct | GenerateImage
    Next: ProductListed | ImageUploaded

Read models:
  - Name: ProductListingModel
    BelongsTo: CreateProduct
```
