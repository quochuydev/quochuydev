### 2. 🟠 Event Storming (Backend Input)

```yaml
Actors:
  - Name: AdminUser
    Role: Admin
    Description: User responsible for managing product listings in the Shopify app.

Commands:
  - Name: CreateNewProduct
    TriggeredBy: AdminUser
    Pre: none
    Next: ProductCreatedEvent

  - Name: GenerateImage
    TriggeredBy: AdminUser
    Pre: none
    Next: ImageGeneratedEvent

Events:
  - Name: ProductCreatedEvent
    Pre: CreateNewProduct
    Next: none

  - Name: ImageGeneratedEvent
    Pre: GenerateImage
    Next: none

Policies:
  - Name: ProductCreationPolicy
    Notes: Ensures all required fields are submitted before creating the product.
    Pre: CreateNewProduct
    Next: ProductCreatedEvent

External systems:
  - Name: ShopifyAdminAPI
    Pre: CreateNewProduct
    Next: ProductCreatedEvent

Read models:
  - Name: NewProductsReadModel
    BelongsTo: CreateNewProduct
```
