    Description: User responsible for managing and creating new textile products in Shopify

  - Name: GenerateProduct
    TriggeredBy: AdminUser
    Pre: none
    Next: ProductGeneratedEvent

  - Name: ProductGeneratedEvent
    Pre: GenerateProduct
    Next: ShopifyApi

  - Name: ProductCreationPolicy
    Notes: Ensures that the product meets all criteria before being created in Shopify.
    Pre: none | ProductGeneratedEvent
    Next: ShopifyApi

  - Name: ShopifyAPI
    Pre: none | ProductGeneratedEvent
    Next: GenerateProduct

  - Name: ProductReadModel
